Add custom buttons to the main menu within SMF

[font=Times New Roman][color=#cc6633][size=3][b]Introduction:[/b][/size][/color][/font]
[hr]
This allows custom buttons to be added to the main menu within SMF at will, complete with children and grandchildren menus (so, a main menu item, a dropdown and a follow-on dropdown)
[list]
[li]Setup this mod at Administration Center » Configuration » Ultimate Menu.[/li]
[li]Should "just work" with all other mods that add their own buttons, such as portals.[/li]
[li]Dynamic by design -- any menu button can change its text without a problem -- no special code required.[/li]
[/list]

[font=Times New Roman][color=#cc6633][size=3][b]Compatibility notes:[/b][/size][/color][/font]
[hr]
[list]
[li]Incompatible with [url=http://custom.simplemachines.org/mods/index.php?mod=3310]Menu Editor Lite / Pro[/url].[/li]
[li]Plays very nicely with [url=http://custom.simplemachines.org/mods/index.php?mod=2650]Add or Remove buttons[/url].[/li]
[/list]
