Add custom buttons to the main menu within SMF

[font=Times New Roman][color=#cc6633][size=3][b]Introduction:[/b][/size][/color][/font]
[hr]
This modification gives the ability to add custom buttons to the main menu within SMF, complete with children and grandchildren menus (main menu item --> dropdown --> additional dropdown)
[list]
[li]Setup this mod at Administration Center » Configuration » Ultimate Menu.[/li]
[li]Should "just work" with all other mods that add their own buttons, such as portals.[/li]
[li]Dynamic by design -- add menu buttons with unique text and icons -- no special code required.[/li]
[/list]

[font=Times New Roman][color=#cc6633][size=3][b]Compatibility notes:[/b][/size][/color][/font]
[hr]
[list]
[li]Only compatible with the [url=https://download.simplemachines.org]SMF 2.1 branch[/url] (SMF 2.0 branch support has been dropped)[/li]
[li]PHP should be compiled with the [GD library or Imagick library] to take advantage of automatic image resizing[/li]
[/list]
